# megi-claude-code-project

Claude Code 스킬 실습 프로젝트입니다.

## 서브 프로젝트

| 프로젝트 | 설명 | 스택 |
|---|---|---|
| [todo-dashboard](./todo-dashboard/) | 할 일 관리 대시보드 앱 | Vite + React + TypeScript + Tailwind CSS v4 |
| [disaster-control](./disaster-control/) | 재난 스트림 API 제어 페이지 (태블릿용) | HTML + Node.js |

## 시작하기

```bash
# todo-dashboard 실행
cd todo-dashboard
npm install
npm run dev

# disaster-control 실행
cd disaster-control
node server.js
```
